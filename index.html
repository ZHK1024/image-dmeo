使用方法

三步完成图片裁剪

自由裁剪

  1. 点击左侧「自由裁剪」按钮切换模式。
  2. 点击「上传图片」或将图片拖入画布区域。
  3. 拖动裁剪框的边角或边线，调整裁剪区域。
  4. 点击右侧「下载图片」导出裁剪结果。

网格裁剪

  1. 点击左侧「网格裁剪」按钮切换模式。
  2. 在「网格设置」中填写行数与列数（如 3×3）。
  3. 上传图片后，画布自动显示等分网格线。
  4. 点击「下载图片」批量导出所有分割子图。
高级用法

通过 URL 参数直接打开并裁剪图片

本工具支持通过页面 URL 的 usource 参数传入图片地址，打开页面时会自动加载该图片，无需手动操作，方便从第三方平台、脚本或浏览器书签直接跳转进行裁剪。

URL 格式

https://你的域名/?usource=图片URL地址

示例

https://你的域名/?usource=https://example.com/photo.jpg

打开上述链接后，工具会自动将图片加载到画布中，可直接开始裁剪操作。

使用步骤

  1. 将目标图片的完整 URL 编码后拼接到 ?usource= 参数后面。
  2. 在浏览器中打开拼接好的链接，页面会自动加载该图片。
  3. 选择「自由裁剪」或「网格裁剪」模式，调整裁剪区域。
  4. 点击「下载图片」完成导出。

注意：目标图片需允许跨域访问（CORS），否则浏览器安全策略可能阻止加载。

常见问题

FAQ

这个工具完全免费吗？

是的，完全免费，无需注册账号，打开网页即可直接使用，没有任何功能限制。

图片会上传到服务器吗？

不会。所有图片处理均在浏览器本地完成，图片数据不会发送到任何服务器，请放心使用。

支持哪些图片格式？

支持 JPG/JPEG、PNG、WebP（静态帧） 等主流格式。

网格裁剪最多可以分割多少块？

行数和列数各最多设置 10，即最多将图片分割为 10×10 = 100 个区块。

裁剪后的图片质量会下降吗？

工具采用高质量导出方式，在保持原始分辨率的前提下进行裁剪，不会产生明显的质量损失。

通过 URL 参数加载图片失败怎么办？

多数情况是因为目标图片服务器未开启跨域（CORS）访问权限。建议先将图片下载到本地，再通过上传方式加载。

手机上能用吗？

支持手机和平板浏览器访问，页面已针对移动端进行适配，可在移动设备上正常使用。
